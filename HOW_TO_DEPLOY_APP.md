# 🚀 Deploy as Earth Engine App (Permanent Link)

Get a **permanent, shareable link** that's always live — no need to copy-paste URLs!

---

## 📋 Option 1: Deploy as Public App (5 min)

### **Step 1: Copy App Script**
File: `gee_app_interactive.js`

### **Step 2: Go to GEE Code Editor**
https://code.earthengine.google.com

### **Step 3: Create New Script**
Click **+ New Script**

### **Step 4: Paste Code**
Copy all of `gee_app_interactive.js` → Paste into editor

### **Step 5: Click "Apps" Button**
Top menu → **Apps**

### **Step 6: Deploy**
Click **Deploy** → Select "Publish new"

**Fill in:**
- Title: `Thailand Daily ET, GPP, WUE 2026`
- Description: `Interactive maps of daily ET, GPP, and WUE for Thailand using MODIS and ERA5 data`
- Click **Publish**

### **Step 7: Get Shareable Link**
After ~30 seconds, you get a **permanent URL:**
```
https://ee-app-XXXXXXXXX.earthengine.app/
```
